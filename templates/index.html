Password generator

Usage: curl -L https://pwgen.btmn.dev/X/Y

Where X is the length of the password and Y is number of passwords you need. This last value is optional and by default it will return one.

CLI Examples

    $ curl -L pwgen.btmn.dev/10
    uWx4q4~JQa

    $ curl -L pwgen.btmn.dev/20/4
    oDMSVb8Qhb.T6p0FehCU
    hL6gXjivTrvqXw&qdN4j
    KFcB0VW%2k7AI7noENRe
    pnta6A1Yacnbh.qpkw5P
