Password generator

Usage: curl -L https://pwgen.zsh.io/X/Y

Where X is the length of the password and Y is number of passwords you need. This last value is optional and by default it will return one.
You can also use the route /vault to request vault-format token IDs.

CLI Examples

    $ curl -L pwgen.zsh.io/10
    VvI2FMUO]1

    $ curl -L pwgen.zsh.io/20/4
    )Zb264P4JsV5Pe7JAjKw
    oJNQBvz9sRumAgZ~if4b
    KvZT(zhceryJyvya34gd
    S\c152niwkRAVyT3bUwO

    $ curl -L pwgen.zsh.io/vault
    wxg379a2-72d6-iay7-mtpq-zqyxdxsqud23
